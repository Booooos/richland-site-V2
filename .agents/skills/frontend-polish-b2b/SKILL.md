# Frontend Polish B2B

## Purpose
Use this skill to refine the visual presentation of an existing B2B export/manufacturer website.

This skill is for polishing, not reinventing.

## Read first
- Read `AGENTS.md`
- Read `DESIGN.md`
- Audit the current page before changing code

## Core objective
Make the interface feel cleaner, more premium, and more intentional through typography and layout refinement.

## What to improve
- Headline scale
- Headline line breaks
- Paragraph readability
- Section spacing rhythm
- Card padding and alignment
- CTA button balance
- Label / metadata consistency
- Desktop and mobile text presentation

## Typography rules
- Prefer strong but controlled heading hierarchy
- Avoid awkward multi-line headline wrapping
- Keep paragraphs readable with comfortable line length
- Use calmer, more refined body text presentation
- Maintain consistent font sizing across similar components
- Avoid excessive font-weight variety in one section
- Improve spacing between headings, paragraphs, proof points, and CTAs

## Layout rules
- Keep section spacing consistent
- Improve column alignment
- Normalize card padding and internal spacing
- Make CTA rows feel balanced and intentional
- Reduce visual clutter
- Preserve mobile readability and tap targets

## B2B design constraints
- Do not turn the site into a SaaS dashboard
- Do not make it overly artistic or experimental
- Do not overuse animation
- Keep product and inquiry information clear
- Keep business facts intact unless small copy edits improve clarity
- Prioritize trust, scanability, and inquiry conversion

## Avoid
- Decorative redesigns
- Concept-page aesthetics
- Repeating the same business keywords too often
- Oversized headings that dominate the page without adding clarity
- Dense blocks of text
- Fancy styling that weakens buyer usability

## Workflow
1. Audit typography and spacing issues
2. Improve heading scale and line breaks
3. Refine paragraph width, line height, and readability
4. Improve card spacing and CTA alignment
5. Tighten section rhythm
6. Check mobile readability
7. Keep business structure intact
