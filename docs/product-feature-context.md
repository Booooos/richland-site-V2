# Product Feature Context

## Scope
This document stores product traits explicitly confirmed from brochure images and current catalog files used in this repository.
Only use features that are clearly shown in brochure text, model-linked icons, product names, category data, or catalog asset naming.
Do not add inferred, assumed, or invented functions, specifications, or claims.

## Data Rules
- Group products by series.
- Record products by model.
- Keep entries short and keyword-based.
- Treat color differences under the same model as color variants.
- Use only traits explicitly confirmed in the available source materials.
